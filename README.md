# AI Smart Crisis Response System

## Overview
AI-based evacuation system with real-time routing and alerts.

## Features
- 3D evacuation simulation
- AI routing & rerouting
- Fire & smoke simulation
- Real-time messaging using Twilio

## How to Run

### Backend
python app.py

### Frontend
Open index.html in browser

## Setup (IMPORTANT)
Create a .env file using .env.example

Add your Twilio credentials:

TWILIO_ACCOUNT_SID  
TWILIO_AUTH_TOKEN  

## Tech Used
- Three.js
- Flask
- Twilio API
